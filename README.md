# 🎬 Netflix Data Analysis (EDA Project)

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on the Netflix dataset to uncover insights about content distribution, trends over time, genres, and country-wise production.

The goal is to understand how Netflix’s content library has evolved and what factors influence content availability.

---

## 🎯 Objective
- Analyze distribution of Movies vs TV Shows  
- Identify trends in content release over the years  
- Explore top genres and categories  
- Understand country-wise content contribution  
- Derive insights for content strategy  

---

## 📊 Dataset Information
- Dataset contains information about Netflix Movies and TV Shows  
- Features include: title, type, director, cast, country, release year, rating, duration, genre  
- Includes both categorical and numerical data  
- Contains missing values handled during preprocessing  

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔍 Steps Performed
1. Data Loading and Inspection  
2. Data Cleaning and Handling Missing Values  
3. Feature Understanding and Transformation  
4. Univariate Analysis (distribution of variables)  
5. Bivariate Analysis (relationships between variables)  
6. Data Visualization  

---

## 📈 Key Insights
- Majority of content on Netflix is **Movies compared to TV Shows**  
- Significant growth in content addition observed after 2015  
- Certain countries contribute more content to Netflix library  
- Popular genres dominate the platform’s content strategy  
- Ratings distribution shows target audience preferences  

---

## 📌 Conclusion
The analysis reveals strong growth in Netflix content over time, with a clear dominance of movies and specific genres. These insights can help in understanding content trends and user preferences.
